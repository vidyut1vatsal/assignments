# 🚀 MSBuild (.NET) Assignment Solution

## 📌 Objective

Is assignment ka goal hai .NET CLI (MSBuild internally use hota hai) ke through ek console application create karna, test project add karna, dependencies install karna, aur build + test + publish karna.

---

## 🛠️ Step 1: Install .NET SDK

### Update system

```bash
sudo apt update
```

### Install .NET SDK

```bash
sudo apt install -y dotnet-sdk-9.0
```

### Verify installation

```bash
dotnet --version
```

### ✅ Expected Output

* .NET SDK version show hoga

---

## 🏗️ Step 2: Create Console App

```bash
dotnet new console -o MyApp
cd MyApp
```

### 📌 Explanation

* `console` → console application template
* `-o MyApp` → project folder name

### 📁 Project Structure

```
MyApp/
 ├── MyApp.csproj
 └── Program.cs
```

---

## 🧪 Step 3: Create Test Project (xUnit)

```bash
dotnet new xunit -o MyApp.Tests
cd MyApp.Tests
```

### Add reference to main project

```bash
dotnet add reference ../MyApp/MyApp.csproj
```

---

## 📦 Step 4: Add Test Packages

```bash
dotnet add package xunit
dotnet add package xunit.runner.visualstudio
```

---

## 💻 Step 5: Add Source Code

Edit file:

```
MyApp/Program.cs
```

```csharp
using System;

namespace MyApp
{
    public class Program
    {
        public static void Main(string[] args)
        {
            Console.WriteLine("Hello MSBuild!");
        }

        public static int Add(int a, int b)
        {
            return a + b;
        }
    }
}
```

---

## 🧪 Step 6: Add Test Code

Edit file:

```
MyApp.Tests/UnitTest1.cs
```

```csharp
using Xunit;
using MyApp;

namespace MyApp.Tests
{
    public class UnitTest1
    {
        [Fact]
        public void TestAddition()
        {
            Assert.Equal(5, Program.Add(2, 3));
        }
    }
}
```

---

## 🔨 Step 7: Build Project

```bash
dotnet build
```

### 📌 Output

* Project compile hoga
* MSBuild internally run hota hai

---

## 🧪 Step 8: Run Tests

```bash
dotnet test
```

### 📌 Expected Output

```
Passed! 1 test
```

---

## 📦 Step 9: Publish (Package)

```bash
dotnet publish -c Release
```

### 📌 Output

* Executable files generate honge
* `bin/Release/net9.0/publish/` folder me output milega

## 🚀 Important .NET CLI Commands Cheat Sheet

| Command              | Purpose              |
| -------------------- | -------------------- |
| `dotnet new console` | Create console app   |

| `dotnet test`        | Run tests            |
| `dotnet run`         | Run application      |
| `dotnet publish`     | Create release build |
| `dotnet add package` | Add dependency       |
