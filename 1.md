# 🚀 Maven Assignment Solution

## 📌 Objective

Is assignment ka goal hai Maven use karke ek Java project create karna, usme dependencies add karna, test likhna, aur project build + package karna.

---

## 🛠️ Step 1: Install Maven & Java

### Update system

```bash
sudo apt update
```

### Install JDK

```bash
sudo apt install default-jdk -y
```

### Install Maven

```bash
sudo apt install maven -y
```

### Verify installation

```bash
mvn -version
```

### ✅ Expected Output

* Java version show hoga
* Maven version show hoga

---

## 🏗️ Step 2: Create Maven Project

```bash
mvn archetype:generate \
-DgroupId=com.example \
-DartifactId=myapp \
-DarchetypeArtifactId=maven-archetype-quickstart \
-DinteractiveMode=false
```

### 📌 Explanation

* **groupId** → package name (company/project)
* **artifactId** → project name
* **archetype** → template (basic Java project)

### 📁 Project Structure

```
myapp/
 ├── pom.xml
 └── src/
     ├── main/java/com/example/App.java
     └── test/java/com/example/AppTest.java
```

---

## 📦 Step 3: Add Dependencies (JUnit 5)

Open `pom.xml` and add:

```xml
<dependencies>
    <dependency>
        <groupId>org.junit.jupiter</groupId>
        <artifactId>junit-jupiter-engine</artifactId>
        <version>5.10.0</version>
        <scope>test</scope>
    </dependency>
</dependencies>
```

---

## ⚙️ Step 4: Configure Surefire Plugin

Add inside `<build>`:

```xml
<build>
  <plugins>
    <plugin>
      <groupId>org.apache.maven.plugins</groupId>
      <artifactId>maven-surefire-plugin</artifactId>
      <version>3.0.0-M7</version>
    </plugin>
  </plugins>
</build>
```

---

## 💻 Step 5: Add Source Code

Edit file:

```
src/main/java/com/example/App.java
```

```java
package com.example;

public class App {
    public static void main(String[] args) {
        System.out.println("Hello Maven!");
    }

    public static int add(int a, int b) {
        return a + b;
    }
}
```

---

## 🧪 Step 6: Add Test Code

Edit file:

```
src/test/java/com/example/AppTest.java
```

```java
package com.example;

import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class AppTest {

    @Test
    void testAddition() {
        assertEquals(5, App.add(2, 3));
    }
}
```

---

## 🔨 Step 7: Compile Project

```bash
mvn compile
```

### 📌 Output

* Java files compile ho jayenge
* `.class` files `target/` folder me generate hongi

---

## 📦 Step 8: Package Project (JAR)

```bash
mvn package
```

### 📌 Output

* `target/myapp-1.0-SNAPSHOT.jar` create hoga

---

## 🧪 Step 9: Run Tests

```bash
mvn test
```

### 📌 Expected Output

```
Tests run: 1, Failures: 0, Errors: 0
```

## 🚀 Important Maven Commands Cheat Sheet

| Command       | Purpose               |
| ------------- | --------------------- |
| `mvn compile` | Compile code          |
| `mvn test`    | Run tests             |
| `mvn package` | Create JAR            |
| `mvn clean`   | Delete target folder  |
| `mvn install` | Local repo me install |
