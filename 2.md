# 🚀 Gradle Assignment Solution

## 📌 Objective

Is assignment ka goal hai Gradle use karke Java project create karna, dependencies add karna, test likhna, aur build + package karna.

---

## 🛠️ Step 1: Install Gradle & Java

### Update system

```bash
sudo apt update
```

### Install JDK

```bash
sudo apt install default-jdk -y
```

### Download Gradle

```bash
wget https://services.gradle.org/distributions/gradle-8.10-bin.zip
```

### Extract Gradle

```bash
sudo mkdir /opt/gradle
sudo unzip -d /opt/gradle gradle-8.10-bin.zip
```

### Set PATH

```bash
export PATH=$PATH:/opt/gradle/gradle-8.10/bin
```

### Verify installation

```bash
gradle -v
```

### ✅ Expected Output

* Gradle version show hoga
* Java version show hoga

---

## 🏗️ Step 2: Create Gradle Project

```bash
gradle init --type java-application -n myapp
```

### 📌 Explanation

* `java-application` → Java app template
* `-n myapp` → project name

### 📁 Project Structure

```
myapp/
 ├── build.gradle
 ├── settings.gradle
 └── src/
     ├── main/java/
     └── test/java/
```

---

## 📦 Step 3: Add Dependencies (JUnit 5)

Open `build.gradle` and update:

```groovy
dependencies {
    testImplementation 'org.junit.jupiter:junit-jupiter:5.10.0'
}

test {
    useJUnitPlatform()
}
```

---

## 💻 Step 4: Add Source Code

Edit file:

```
src/main/java/App.java
```

```java
public class App {
    public static void main(String[] args) {
        System.out.println("Hello Gradle!");
    }

    public static int add(int a, int b) {
        return a + b;
    }
}
```

---

## 🧪 Step 5: Add Test Code

Create/Edit file:

```
src/test/java/AppTest.java
```

```java
import org.junit.jupiter.api.Test;
import static org.junit.jupiter.api.Assertions.*;

class AppTest {

    @Test
    void testAddition() {
        assertEquals(5, App.add(2, 3));
    }
}
```

---

## 🔨 Step 6: Build Project

```bash
./gradlew build
```

### 📌 Output

* Project compile hoga
* Tests run honge
* `build/` folder create hoga

---

## 🧪 Step 7: Run Tests

```bash
./gradlew test
```

### 📌 Expected Output

```
BUILD SUCCESSFUL
```

---

## 📦 Step 8: Package Project (JAR)

```bash
./gradlew jar
```

### 📌 Output

* `build/libs/myapp.jar` create hoga
